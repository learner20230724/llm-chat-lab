# llm-chat-lab MVP

## First slice

Build a small web app with:
- two-column compare view
- shared input box
- independent prompt preset per panel
- independent model preset per panel
- local saved sessions
- simple metrics row: latency, token estimate, cost estimate

## Success bar

A stranger should be able to open the repo, look at the screenshot or demo, and understand in under 15 seconds why this is more useful than a basic chat box.
