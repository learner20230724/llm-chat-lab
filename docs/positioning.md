# llm-chat-lab positioning

## Core statement

`llm-chat-lab` is a local-first workspace for comparing how chat systems behave under different prompts, model presets, memory policies, and workflow wiring.

## Who it is for

- developers building LLM chat products
- prompt / workflow tinkerers
- people comparing model behavior across setups
- anyone who wants to inspect, not just chat

## What makes it distinct

- compare-first, not single-thread-first
- local-first by default
- visual enough to demo publicly
- useful even with mock mode or a limited provider setup

## Non-goals

- replacing full chat clients
- pretending to be a universal agent platform
- hiding the tradeoffs behind marketing language
